Instalação GIT

Baixe o arquivo executável do instalador a partir da página do GitHub e execute-o:

http://msysgit.github.com

Siga os passos conforme as imagens abaixo, não será necessário alterar nenhuma opção durante a instalação.

Após concluir a instalação, você terá tanto uma versão command line (linha de comando, incluindo um cliente SSH que será útil depois) e uma GUI padrão.
