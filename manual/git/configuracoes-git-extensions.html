Configurações iniciais Git Extensions

  1. Abrir o git-extensions e selecionar “Tools > Settings”. Deve aparecer uma lista com todas as configurações básicas para o git-extensions funcionar corretamente. Observando a imagem, só falta informar nome de usuário e e-mail, para informa-los clique sobre o botão “Repair” ao lado do bloco laranja ou “Global settings” no menu do  lado esquerdo.
  2. Informe o nome de usuário e o e-mail cadastrado no github.
  3. Selecione “SSH” no menu ao lado esquerdo, marque a opção “PuTTY”..
  4. Volte a opção “Git Extensions” no menu do  lado esquerdo. Agora a lista de configurações deve estar com todos os itens na cor verde.

Nota: No Windows 7 e Windows 8, verifique o arquivo .gitconfig na sua pasta de usuário.

Para abrir o arquivo, vá em executar (Win + R) e digite “ 

%HOMEDRIVE%%HOMEPATH%\.gitconfig

Procure a linha com o seguinte conteúdo: 

[credential]
helper = !\\\"C:/Program Files (x86)/GitExtensions/GitCredentialWinStore/git-credential-winstore.exe\\\"

E altere para: 

[credential]
helper = !\"C:/Program Files (x86)/GitExtensions/GitCredentialWinStore/git-credential-winstore.exe\"
