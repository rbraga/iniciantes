Barra de Ferramentas (Visual Studio)

  1. Commit : Realiza um commit local no branch que é demonstrado, na imagem acima o branch é o master.
  2. Browse repository : Visualiza o repositório, seus commits, branchs, tags.
  3. Pull : Trás modificações realizadas no repositório remoto para o repositório local.
  4. Push:  Envia commits realizados no repositório local para o repositório remoto.
  5. Stash:  Guarda alterações que foram feitas, mas que ainda não estão em um commit, para restauração posterior.
  6. Settings:  Abre janela com opções do git-extensions.
