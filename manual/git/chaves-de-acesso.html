Chaves de Acesso

Falta somente configurar as chaves de acesso ao github, para gerar uma chave.

  1. No git-extensions selecione “Tools > Putty > Generate or import key”.
  2. Clique em “Generate” para gerar uma chave. (Durante a geração, mova o mouse sobre o quadro “Key”).
  3. Após gerar a chave, informe uma senha no campo “Key passphrase” e sua confirmação. Salve a chave privada em “Save private key”.
  4. Ainda com a janela com a chave gerada aberta, copie o conteúdo da chave, ela será utilizada no github.
  5. Acesse o endereço https://github.com/settings/ssh  e selecione o botão “Add SSH key”, informe um titulo para a chave, e cole a chave no campo “Key”.

Agora só precisamos vincular a chave gerada no PuTTY e já configurada na conta do github com o git-extensions. Para fazer isso, vamos clonar um projeto do github.

  1. No git-extensions, selecione “Github > Fork/Clone repository” e busque o projeto a ser clonado, no nosso caso será o projeto do jquery, após selecionar o projeto clique no botão “Clone”.
  2. Será solicitada uma chave SSH, selecione a opção “Load SSH key” e selecione a chave salva.
  3. Se foi definida senha para a chave, informe a mesma.
